# 🌍 Wanderlust

> 🚧 **Note:** This project is still under active development. Some features may not be fully implemented yet and could be incomplete. Future updates will enhance functionality.

**Wanderlust** is a full-stack travel accommodation web application inspired by Airbnb(Clone). It offers a secure and scalable platform where users can browse listings, register/login, and manage bookings. Built with **Node.js**, **EJS**, **MongoDB**, and modern CSS, it follows a clean design and modular architecture.  

The project is **deployed on Render** and uses **MongoDB Atlas** as the cloud database for global accessibility and reliable performance.

## 🔐 Key Features

- 🔑 Full authentication with session-based login/logout  
- 📝 Secure registration & form validation  
- 🛠 Modular backend with Express.js & EJS templates  
- ☁️ Cloud database using MongoDB Atlas  
- 📱 Responsive UI with reusable components  
- ⚠️ Error handling & flash messaging  
- 🚀 Deployed on Render  

---

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3, EJS, Font Awesome  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas + Mongoose  
- **Auth & Security:** Passport.js, express-session, connect-flash  
- **Deployment:** Render (server) + MongoDB Atlas (database)
